# Google Mobile Ads Next Gen SDK for Android

The GMA Next Gen SDK enables mobile app developers to maximize their monetization in native
mobile apps.

This repository contains open source examples and developer resources for the GMA Next Gen SDK,
covering both the Google AdMob and Google Ad Manager components of the Google Mobile Ads SDK.

# Google Mobile Ads SDK Developers forum

To get developer support, visit the [Google Mobile Ads SDK Developers forum](https://groups.google.com/forum/#!forum/google-admob-ads-sdk).

# Discord
